# 周复盘

一个关于如何做好周复盘的分享页面。

---

## 灵感来源

很多人每周复盘时只会说「这一周好忙」，却说不出具体忙了什么、有什么收获、下周要怎么改进。

这个页面尝试用更结构化的方式，引导你做一次真正有价值的周复盘。

## 内容

- 为什么「这一周好忙」不是复盘
- 如何清晰地描述本周进展
- 如何从失败中提取教训
- 如何制定可执行的下周计划

## 三种复盘方式

周复盘不是只翻日历，而是用三种方式交叉验证：

### 1. 口喷
先说出来，把感受释放出来。不要先冲进日历统计小时数，而是先问自己：这周做了什么？什么让我很有劲，什么让我很累？哪些事情卡住了？

记录的是：**我怎么经历这一周**

### 2. 查日历
看真实发生了什么。日历不能解释动机，但它很诚实——它会告诉你时间花在哪里。

记录的是：**我实际上做了什么**

### 3. 查 OB 会议记录和笔记
看成果和长期方向。一件看上去很努力的事，也可能没有真正推进任何结果。

记录的是：**我正在走向哪里**
